臺中社宅候補查詢工具｜17租公開候補名單與遞補進度

這是一個非官方的臺中社宅候補查詢工具，整理臺中市 17租公開候補名單， 方便查詢臺中社會住宅各案場、遞補類型、房型、戶別與候補序號變化。 使用者也可以上傳自己的 my_applications.csv，或直接在網頁中輸入多筆候補資料， 進行個人候補狀態與遞補進度估算。

資料來源

本工具整理臺中市社會住宅相關公開資料與 17租公開候補資訊， 目的是讓候補資料更容易查詢與比較。

重要提醒： 本工具不是臺中市政府官方網站，查詢與估算結果僅供參考。 實際申請資格、候補通知、選屋、入住時間與最新公告， 請以臺中市政府住宅發展工程處及 17租官方系統為準。 使用者上傳或輸入的個人候補資料僅供當次查詢使用，本工具不提供帳號登入，也不會將資料寫入 GitHub 或公開資料庫。
